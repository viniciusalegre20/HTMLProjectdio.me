Ferramentas utilizadas

As ferramentas utilizadas para produção de um código em HTML podem ser desde um editor de texto qualquer, como notepad do próprio windows, até uma IDE mais completa, como o VisualStudioCode, em que podem ser instaladas diversas extenções para aprimorar a forma em que a construção do código é feita.

Algumas das extenções que foram usadas para a criação deste site foram:

  • Emmet- Função de preenchimento automático das sintaxes
  • Live server- Atualização automática da página que está sendo modificada

Inspetor de Elementos - Browser

O inspetor de elementos de um browser é utilizado para inspecionar uma página e observar como essa página foi idealizada e criada, podendo assim visualizar a forma utilizada para a criação da mesma. Para utilizar essa ferramenta, existem 2 métodos diferentes, apertando a tecla F12 do teclado, ou clicando com o botão direito no corpo do site e selecionando a opção inspecionar.

Estrutura Básica do HTML - Onde tudo se organiza

Sendo elas sempre iniciadas utilizando a abertura do simbolo menor (<) e o simbolo maior (>) para o fechamento das estruturas. Nem sempre essas estruturas precisam do fechamento das suas tags, caso precisem utilizaremos do seguinte modo: </>
O HTML tem sua estrutura baseada em 3 partes.

  • Identificação
  • Metadados
  • Corpo da página

A estrutura de identificação é o local onde o browser irá identificar o início da linguagem escolhida.
No nosso caso utilizamos a sintaxe:

<html> </html>

Essas duas tags, de início e fechamento estarão no começo e no final do nosso código.


A estrutura de metadados é uma área selecionada para que o browser processe informações que não apareceram no corpo da página.
Sendo assim, tudo que será colocado no escopo de metadados o usuário não conseguirá ver em sua tela e são utilizados para modificarem a página por completo.
Sua sintaxe consiste na seguinte forma:

<head> </head>


No head é definido diversos tipos de função, podendo alterar o estilo da página, formatação, etc. Alguns exemplos são:

<meta charset="UTF-8"> - Definição da cadeia de caracteres disponíveis para utilização dentro da página.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> - Responsividade da página dependendo do dispositivo utilizado, se adaptando a resolução do aparelho.

<title> </title> - Mudança to título inicial da página.


O corpo da página é definido completamente dentro da tag <body>, tudo o que o usuário vê em sua tela, será colocado dentro dela.
Assim teremos a seguinte sintaxe:

<body> </body>

Nesta parte do código serão inseridas tags de texto e de estruturamento, para definir diferenças entre parágrafos, cabeçalhos, imagens, links e etc.

Links

Para a criação dos hyperlinks utlizamos a nomenclatura <a>, e é com ela que conseguimos redirecionar o usuário até outro site, ou até mesmo para o nosso próprio site, porém em outra página totalmente diferente da anterior.
A sintaxe é um pouco mais complexa das vistas anteriormente.Sendo ela da seguinte forma:

<a href="link do website">"texto que irá redirecionar a outra página"</a>
Os links não necessáriamente precisam estar em uma texto para que redirecione o usuário a outa página, tabém é possivel utilizar uma imagem.
Exemplo:

<a href="google.com"><img src="google.png" alt="google"></a>

Com essas informações que vimos anteriormente, ao clicar em um link,seremosS redirecionados a outra página, mas sem entrar em outra janela do navegador. O que podemos fazer para que isso não ocorra ?
Para evitar isso utilizamos o atributo target="_blank", assim o browser abrirá uma nova janela no navegador. Exemplo:

<a href="google.com" target="_blank"><img src="google.pnc" alt="google"></a>
